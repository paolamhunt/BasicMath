This should be done without using a loop. Finally, sort the DataFrame by BA, in descending order. Use head() to display the first 10 rows of this DataFrame.

# Create a DataFrame named batting_avg
# Select teamID, yearID, H, and AB columns from the batting DataFrame
# Group results by teamID and yearID
# Calculate the sum of the remaining two columns within each group. 

batting_avg = batting.loc[:,['teamID','yearID', 'H','AB']
                    ].groupby(['teamID', 'yearID']
                             ).sum()

# Add a new column named BA to the batting_avg DataFrame
# Without using a loop, calculate new col by dividing the values in the H column by the values in the AB column
# Sort the DataFrame by BA, in descending order
batting_avg.loc[:,'BA'] = batting_avg.loc[:,'H'] / batting_avg.loc[:,'AB']
batting_avg=batting_avg.sort_values(by='BA', ascending=False)

# Create a DataFrame named stl_batting
# Use loc to filter batting DF; keeping only SNL team and year >= 1900
# Use loc to select the yearID, H, AB, and HR
# Group the results by yearID
# Calculate grouped sums

stl_sel = (batting.loc[:,'teamID'] == 'SLN') & (batting.loc[:,'yearID'] >= 1900)
stl_batting = batting.loc[stl_sel,['yearID','H','AB','HR']
                         ].groupby('yearID'
                                  ).sum()

# Add a new column named BA to the stl_batting DataFrame
# Without using a loop, calculate new col by dividing the values in the H column by the values in the AB column
# Sort the DataFrame by BA, in descending order
stl_batting.loc[:,'BA'] = stl_batting.loc[:,'H'] / stl_batting.loc[:,'AB']
